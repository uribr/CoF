Photo Filtering

In the bottom of this screen there are 4 buttons:

  • The Scrrible Switch - toggle this switch in order to enable scribble drawing on the image. Draw scribble on foreground objects in order to create a foreground mask for the statistics collection phase.
  • Clear Scrrible - press this button if you want to delete the scribbles you've drawn so far.
  • Settings (the build icon) - press this button to set the different parameters of the filter.
  • Apply Filter - press this button when you're ready to go - if the scribble switch is off, it will run the CoF algorithm using the default mask (collecting co-occurrence statistics from the whole image). If the scribble switch is on, it will create a mask from the scribble. In the next step you will set the mask as tight as you like, and then the FB-CoF algorithm will take place.
